# A music player visualizer
A music player with a library of music to play

## Features
* Pick a song out of a list of songs
* Pause Song
* Seek Song
* Visualizer
* Volume

### Planned Features
* Pick a theme
* Different types of visualizers
    * Box Visualizer
    * Path
* Particles
